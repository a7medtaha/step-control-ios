# Step control iOS

A cool step-by-step indicator control within a pager for iOS.

![Alt Text](https://github.com/moove-it/step-control-ios/blob/master/step-control-ios.gif)

Installation
--------

Via [Cocoapods](https://cocoapods.org/):

```
pod 'StepControliOS', '~> 0.1'
```
